## Installation

Use npm:

```bash
npm install --save dark-light-mode
```

Use yarn:

```bash
yarn add dark-light-mode
```

Use CDN:

```html
<script type="module" src="https://unpkg.com/dark-light-mode"></script>
```

## Usage

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>ğŸŒ’ toggles dark mode... Really? Try it</title>
    <meta charset="utf-8" />
    <script type="module" src="https://unpkg.com/dark-light-mode"></script>
  </head>
  <body>
      <dark-light-mode />
  </body>
</html>
```
